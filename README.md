# Ruest client

## Keymappings

Global keys:
- Quit: `q`
- Toggle request list: `r`

Movement keys:
- Left: `Left arrow` or `h`
- Right: `Right arrow` or `l`
- Up: `Up arrow` or `k`
- Left: `Left arrow` or `j`
- Previous tab: `Shift+Tab`
- Next tab: `Tab`

Actions:
- Exit: `Esc`
- Accept: `Enter` or `Spc`
